## Observations

- `dist/maxc/maxc` is a Mach-O 64-bit `arm64` executable.
- All 3 bin-smoke assertions PASS:
  - `maxc --version` → `maxc 0.2.5`
  - `maxc --help` → exit 0
  - `maxc --format json agent skill` → `skill_exists OK`
- PyInstaller `EXE target arch: arm64`, native ARM64 build (no Rosetta).
- **Bundle size: 234 MB.** This is ~4× the spec's working assumption ("~50MB" in § 4 first-install UX message). Drivers:
  - `pyodps` SDK pulls in a lot of optional deps
  - `_internal/` contains the full Python 3.12 stdlib + transitive packages
- Implication: first-install download time will be longer than estimated. Two follow-ups worth noting (NOT blockers):
  - Update spec § 4 UX message to reflect realistic size (`~230MB` or `1–2 minutes`)
  - Consider PyInstaller `excludes=` to strip unused stdlib modules (tkinter, test, idlelib, etc.) — could halve the bundle. Defer to a separate optimization task.
